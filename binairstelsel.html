Kort uitgelegd: binaire code wordt gebruikt om instructies te geven aan computers met behulp van maar twee symbolen, die symbolen zijn 1 en 0.(lange uitleg onder het spel)

ASCII staat voor:American Standard Code for Information Interchange. Dat betekent: Amerikaanse standaardcode voor informatie-uitwisseling. Het is een standaard code voor ASCII tekens, ze kunnen op elk soort computers worden gebruikt. Elk ASCII teken heeft een nummer dat voor alle computers hetzelfde is.

ASCII table

Binair betekent tweecijferig, het wordt zo genoemd omdat je in het binairstelsel met twee cijfers, 0 en 1, alle getallen moet kunnen opschrijven. Het woord bit is een samenvoeging van de woorden "binary" en "digit". Een bit is het kleinste deeltje van een Byte, een Byte is een deeltje van 8 bits naast elkaar van 1 en 0.
